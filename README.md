# Bspwm

Configura tu propio entorno de linux con Bspwm.

## Kali Linux

```
git clone https://github.com/bl4ck44/Bspwm.git

cd Bspwm

cd Kali-Linux

sudo chmod +x install.sh

cd ..

bash setup.sh
```

## Parrot OS

```
git clone https://github.com/bl4ck44/Bspwm.git

cd Bspwm

sudo chmod +x install.sh

```
